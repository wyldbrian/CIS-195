Example Elements (Structural)

The Address Element

The address element can provide contact information for a single page, or an entire site

<address>
	John Smith
	1234 Somewhere Rd.
	Bend,OR 97701
	USA
</address>

The Section Element

The section element is used to organize content into sections. Think of the difference between an introduction, body, and conclusion.

<section>
	<h1>The Great War</h1>
	<p>During the great war...</p>
</section>

The Article Element

The article element is used to identify content that can stand alone. It is often used for blog posts.

<article>
	<h1>My Vacation!</h1>
	<p>During my 2017 vacation, I visited several awesome places...</p>
</article>

The Header Element

The header element is used for introductory content on your site.

<header>
	<h1>Primary Site topic</h1>
	<h2>Core purpose of the site in additional detail</h2>
	<p>Less important verbose details about the site</p>
</header>

The Nav Element

The nav element is used to identify navigation links.

<nav>
	<a href="/about/">About</a>
	<a href="/contact/">Contact</a>
	<a href="/news/">News</a>
</nav>

Example Elements (Phrasing)

The Mark Element

The mark element is used to highlight relevant text.

Please understand, <mark>this must remain private!</mark>

The Alternatice Voice Element (i)

The alternative voice element italicizes text to suggest a different mood.

He goes by <i>James Bond</i>.

The Small Element

The small element provides extremely small print (think terms & conditions)

<small>If you break this, you're screwed.</small>

The Strikethrough Element

The strikethrough element puts a line through your text indicating it is no longer relevant.

We're currently operation at 90% capacity, <s>and we'll need an additional 20 server racks.</s>

The Importance Element (Strong)

The importance element makes text bold to show that it is more important than the surrounding text.

Please remember, <strong>you must be back by 10PM!</strong>
