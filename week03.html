Hyperlinks

A relative hyperlink would be a hyperlink to another page within the same web site. My site, I've got pages such as index, practice, and week03b.

Absolute hyperlinks are hyperlinks that link to pages (or sites) outside of your own web site. For example hyperlink to eBay, I would use an absolute reference.

Bookmark hyperlinks are used when you want to link to a specific part of a web page. For instance, check out eBay's footer. Or, hyperlink to a part of the current page, such as this page's footer.
